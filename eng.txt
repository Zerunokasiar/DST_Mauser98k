[h1]3. Traits[/h1]

[b]Reload[/b]
Action : Reload!
[table]
[tr][td]
Fill ammo clip of equipped gun when reload.
[/td][/tr]
[/table]

[b]Charge[/b]
Action : Charge!
[table]
[tr][td]
When charge, move 40% faster and consume hunger 96% faster.
If hunger is 0 when charge, damage reduce about 50% down.
It has faster attack timing than normal if hunger is not zero when bayonet is equipped. 
[/td][/tr]
[/table]

[b]Instant Fire[/b]
Action : Fire!
[table]
[tr][td]
Rapidly ranged attack when melee mode.
Attack range is short than ranged attack mode, it can be miss when enemy is moving or too close. 
When enemy exist near mouse point, action is displayed.
If controller, when enemy exist in range from self, action is displayed.
[/td][/tr]
[/table]

[b]Change Mode[/b]
Action : Use
[table]
[tr][td]
Switch melee attack mode or ranged attack mode.
Ranged attack mode cannot melee attack, but pre-aiming is faster.
If gun clip is empty, cannot change to ranged attack mode.
[/td][/tr]
[/table]

[b]Repair Bayonet[/b]
Action : Give
[table]
[tr][td]
Repair as durability of provided bayonet.
If durability is limit over, bayonet's durability is only reduce, not removed.
[/td][/tr]
[/table]

[b]Repair Gunstock[/b]
Action : Give
[table]
[tr][td]
Repair as durability of provided gunstock.
If durability is limit over, gunstock's durability is only reduce, not removed.
[/td][/tr]
[/table]

[b]Reload Ammo[/b]
Action : Give
[table]
[tr][td]
Refill gun as stack of provided ammo.
If clip is limit over, remaining ammo is return.
[/td][/tr]
[/table]

[b]The weapon for the weak[/b]
[table]
[tr][td]
In case of ranged attack, damage rate is calibrated to 1.
[/td][/tr]
[/table]

[b]Additional critical hit[/b]
[table]
[tr][td]
Add n/2 damage rate if damage multiplier is over than n when ranged attack.
ex) Damage Rate 1.2 > Additional Damage Rate 0.1
[/td][/tr]
[/table]

[b]Broken Gun[/b]
[table]
[tr][td]
If durability is 0, gun is gone.
When broken, ammo spread near character if gun clip is not empty.
[/td][/tr]
[/table]

[b]Broken Bayonet[/b]
[table]
[tr][td]
If durability is 0, bayonet is broken.
It return to gun without bayonet, gunstock durability and reloaded ammo is preserved.
[/td][/tr]
[/table]
